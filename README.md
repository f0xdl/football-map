# Football Match Analytics Pipeline
Event-driven платформа, которая:
- принимает события матча (голы, удары, фолы, владение мячом)
- обрабатывает их асинхронно
- агрегирует статистику
- хранит её в PostgreSQL
- предоставляет API для медиа-клиентов (сайт, мобильное приложение)

## Goals
Необходимо реализовать event-driven систему аналитики футбольных матчей для медиа-платформы.

Система принимает события матча по HTTP, сохраняет их в PostgreSQL, асинхронно обрабатывает через RabbitMQ, агрегирует статистику и предоставляет API для чтения агрегированных данных.

Решение должно быть контейнеризовано, развёртываться в Kubernetes и иметь полный CI/CD пайплайн в GitLab.

# Services
## ingest-service
REST API для получения событий
## stats-processor
Обработка полученных событий и агрегация статистики
## analytics-api
REST API для медиа-клиентов
